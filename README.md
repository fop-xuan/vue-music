# vue-music 
数据均来源于酷狗音乐 一个练手的项目，如有不足多多包涵

# 主要实现技术：
 - vuejs2.6
 - vue-router
 - less
 - vuex
 - axios
 - cube-ui
 - vue-lazyload 图片懒加载
 - es6 
 - webpack
